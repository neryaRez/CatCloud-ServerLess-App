AWS Serverless Project

🐱 Cat Gallery Notifier

A fully automated serverless application that uploads sample cat files to S3, lists bucket objects with Lambda, and sends SNS email notifications on every execution.

View Architecture Manual Lambda Test

📦 S3 Storage

Sample files are uploaded automatically to an S3 bucket during deployment.

⚡ Lambda Scanner

The Lambda function lists all objects in the bucket and prepares execution details.

📧 SNS Notification

Each Lambda execution publishes an email notification through Amazon SNS.

🚀 GitHub Actions

Infrastructure deployment is triggered manually using workflow_dispatch.

Architecture Flow

GitHub Actions → CDK Deploy → S3 Bucket → Lambda → SNS Email

Manual Lambda Test

After deployment, the Lambda can be manually invoked using the AWS CLI. 

aws lambda invoke \
  --function-name <LambdaFunctionName> \
  --cli-binary-format raw-in-base64-out \
  --payload file://test-events/manual-test-event.json \
  response.json

Project Purpose

This project intentionally keeps the application layer simple and focuses on DevOps fundamentals: Infrastructure as Code, least-privilege IAM, CI/CD, S3 object management, Lambda execution, and SNS notifications.
